# 🧭 TraveLog (트래블로그)

> 잊지 못할 여정의 특별한 조각들을 모으다.
> **TraveLog**는 사용자의 소중한 여행 경험을 시각적이고 직관적으로 기록하고 공유할 수 있는 모던 웹 애플리케이션입니다. 편리한 UI와 감각적인 인터랙션을 통해 당신만의 여행 이야기를 완성해보세요.

---

## ✨ 주요 기능 (Key Features)

* **프리미엄 인트로 스플래시**: 페이지 진입 시 브랜드 로고(나침반)와 프로그레스 바가 흐르는 몰입감 있는 오프닝 연출
* **타이핑 애니메이션 텍스트**: 메인 배너의 핵심 카피가 한 글자씩 나타나는 생동감 있는 효과
* **그라데이션 UI**: 핑크 & 퍼플 톤의 화사하고 모던한 그라데이션 브랜딩 적용
* **반응형 웹 디자인**: 모바일, 태블릿, 데스크톱 등 다양한 디바이스 환경 최적화

## 🛠 기술 스택 (Tech Stack)

### Front-end
* **HTML5 / CSS3**
* **Tailwind CSS** (스타일링 및 그라데이션 효과)
* **Vanilla JavaScript** (스플래시 화면 제어 및 애니메이션 큐잉)
* **Font Awesome** (아이콘 에셋 - `fa-compass` 등)
